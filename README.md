# 📝 Category Todo App

## 🚀 Overview
A category-based todo list application built with vanilla JavaScript. Users can create, edit, delete, and complete tasks within multiple categories. The app uses CRUD operations and local storage to ensure tasks persist even after page refresh.

---

## ✨ Features
- Category-based task organisation  
- Add, edit, delete, and complete tasks  
- Persistent data using local storage  
- Dynamic UI updates without reload  
- Clean state-based architecture  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- SASS
- JavaScript (ES6+)  
- Local Storage API  

---

## 🧠 Key Concepts
- CRUD operations (Create, Read, Update, Delete)  
- Event delegation  
- State management using objects and arrays  
- DOM manipulation  
- Data persistence with local storage
